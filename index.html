Bienvenidos a el Inventario de herramientas y materiales de mantenimiento

Dentro de esta aplicacion web usted podra realizar el control de todas las herramientas y materiales de mantenimiento, para comprender su funcionamiento le dejamos las siguientes instrucciones.

En caso de necesitar ayuda abajo se encuentran los contactos de soporte.

Registros

En la seccion de registros podras visualizar todos los movimientos que se cargaron en la pagina, incluyendo datos como el id de cada registro, la cantidad y categoria de las herramientas y materiales tambien datos como el Nombre y Numero de legajo de la persona que haya retirado o ingresado las mismas.
